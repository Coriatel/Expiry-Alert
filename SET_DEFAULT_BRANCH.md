# How to Set Default Branch

The current branch `claude/reagent-expiration-tracker-d0bju` contains all the code and is ready for production.

## Steps to Make This the Main Branch:

1. **Go to GitHub Repository Settings:**
   - Navigate to: https://github.com/Coriatel/Expiry-Alert/settings

2. **Change Default Branch:**
   - On the left sidebar, click **"Branches"**
   - Under "Default branch", click the switch icon (↔️)
   - Select: `claude/reagent-expiration-tracker-d0bju`
   - Click **"Update"**
   - Confirm by clicking **"I understand, update the default branch"**

3. **Done!** ✅

Now this branch will be:
- ✅ The default when people visit your repo
- ✅ The default for Pull Requests
- ✅ What GitHub Actions builds
- ✅ What shows in README

---

## Alternative: Rename This Branch (If You Have Admin Access)

After setting as default, you can rename it:

1. Go to: https://github.com/Coriatel/Expiry-Alert/branches
2. Click the pencil icon (✏️) next to `claude/reagent-expiration-tracker-d0bju`
3. Rename to: `main` or `master`
4. GitHub will update all references automatically
