# Mobile App Assets

## Required Assets

To build the mobile app, you need to provide:

### Icons
- `icon.png` - 1024x1024 app icon
- `adaptive-icon.png` - 1024x1024 Android adaptive icon
- `favicon.png` - 48x48 web favicon

### Splash Screen
- `splash.png` - 1284x2778 splash screen image

## Generating Assets

You can use online tools or Figma to create these:

1. **App Icon (1024x1024)**
   - Use same design as desktop icon
   - Simple, recognizable medical/lab theme
   - Good contrast for light and dark backgrounds

2. **Splash Screen (1284x2778)**
   - App name + icon
   - Clean white or light background
   - Optional: tagline
