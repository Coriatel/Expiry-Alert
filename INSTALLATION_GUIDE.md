# מדריך התקנה | Installation Guide

## 🖥️ אפליקציית Desktop (Windows)

### הורדה
1. עבור ל-[Releases](../../releases) בGitHub
2. הורד את הקובץ המתאים:
   - `Reagent-Expiry-Tracker_1.0.0_x64_en-US.msi` (מומלץ)
   - או `Reagent-Expiry-Tracker_1.0.0_x64-setup.exe`

### התקנה
1. **פתח את הקובץ שהורדת** (.msi או .exe)
2. **אשר** את ההתקנה (אם Windows Defender מציג אזהרה, לחץ "More info" ו-"Run anyway")
3. **עקוב** אחרי אשף ההתקנה
4. **זהו!** האפליקציה מותקנת ומוכנה לשימוש

### הפעלה
- פתח את האפליקציה מתפריט Start או מסמל שולחן העבודה
- **לא צריך אינטרנט!** - הכל עובד מקומית על המחשב שלך

### מיקום המידע
- כל המידע נשמר ב: `C:\Users\<username>\AppData\Local\reagent-expiry-tracker\reagents.db`
- לגיבוי: פשוט העתק את הקובץ הזה למקום בטוח

## 📱 אפליקציית Mobile (Android)

### הורדה
1. עבור ל-[Releases](../../releases) בGitHub
2. הורד את הקובץ `reagent-tracker.apk`

### התקנה (Android)
1. **הפעל "Unknown Sources"**:
   - הגדרות → אבטחה → אפשר התקנה מ-"Unknown Sources"
   - או אפשר רק עבור הדפדפן שבו הורדת את הקובץ
2. **פתח את קובץ ה-APK** שהורדת
3. **לחץ "Install"**
4. **אשר** את הרשאות הנדרשות
5. **זהו!** האפליקציה מותקנת

### התקנה (iOS)
**שים לב:** התקנה ב-iOS דורשת חשבון Apple Developer או TestFlight.

1. הצטרף ל-TestFlight beta דרך הקישור שתקבל
2. התקן את TestFlight מה-App Store
3. פתח את הקישור ל-Beta
4. התקן את האפליקציה

### הפעלה
- פתח את האפליקציה מהמסך הראשי
- **לא צריך אינטרנט!** - הכל עובד מקומית על הטלפון שלך

## 🔧 פתרון בעיות

### Desktop

**"Windows protected your PC"**
- לחץ "More info"
- לחץ "Run anyway"
- זה קורה כי האפליקציה לא חתומה דיגיטלית (עולה כסף)

**האפליקציה לא נפתחת**
- ודא ש-Windows 10/11 מעודכן
- נסה להריץ כ-Administrator (לחץ ימני → "Run as administrator")

**המידע נמחק אחרי עדכון**
- זה לא אמור לקרות! המידע שמור ב-AppData
- אם זה קרה, חפש גיבוי ב: `C:\Users\<username>\AppData\Local\reagent-expiry-tracker\`

### Mobile

**"App not installed"**
- ודא שאפשרת התקנה מ-"Unknown Sources"
- נסה למחוק גרסה קודמת אם קיימת
- ודא שיש מספיק מקום בטלפון (צריך ~50MB)

**האפליקציה קורסת**
- נסה למחוק ולהתקין מחדש
- ודא שהאנדרואיד הוא גרסה 8.0 ומעלה
